# AADH-YOLOv5
This repository is about the work of AADH-YOLOv5: improved YOLOv5 based on adaptive activate decoupled head for garbage detection. Dataset and models will be publicly available if the paper is accepted.

# Datasets
We provide the datasets DGD-6 of our paper in [BaiDuYunlink](https://pan.baidu.com/s/1JIBtz8AR_kM9eWwd1N9nHQ).
